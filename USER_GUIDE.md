## 🚀 新手上路：5 分鐘快速入門

### 步驟 1️⃣ 登入系統

1. 打開瀏覽器，進入：**https://cagoooo.github.io/comments/**
2. 點擊「**使用 Google 帳號登入**」
3. 選擇您的 **@教育雲帳號** 或 **個人 Gmail**
4. 首次登入需要**填寫申請資料**，等待管理員審核

> 💡 **小提示**：使用個人 Gmail 登入的 API Key 功能會比較穩定喔！

---

### 步驟 2️⃣ 輸入學生名單

#### 方法 A：手動輸入（少量學生）
1. 在左側「**新增學生**」區域輸入姓名
2. 按 **Enter** 或點擊「新增」按鈕
3. 學生就會出現在右邊的表格中！

#### 方法 B：Excel 匯入（整班學生）
1. 準備好 Excel 檔案（第一欄放姓名）
2. 點擊「**📥 Excel 匯入**」按鈕
3. 選擇檔案，一鍵匯入全班！

> ⚠️ **注意**：Excel 格式請確保第一欄是學生姓名

---

### 步驟 3️⃣ 為學生選擇成語標籤

這是最好玩的部分！

1. 點擊右側「**🏷️ 成語標籤**」欄位
2. 左側會展開成語選單
3. 選擇適合該學生的成語（可多選）
4. 常用成語會自動排序到最前面！

**成語分類**：
| 類別 | 範例 |
|------|------|
| 🧠 資賦 | 才思敏捷、舉一反三、觸類旁通 |
| 📖 學業 | 勤學不倦、孜孜不倦、學業進步 |
| 💪 品德 | 彬彬有禮、誠實守信、知錯能改 |
| 🤝 人際 | 樂於助人、與人為善、團結合作 |
| 🙋 服務 | 熱心服務、盡心盡責、任勞任怨 |

---

### 步驟 4️⃣ AI 生成評語（魔法時刻✨）

1. 選好成語標籤後，點擊「**🤖 生成評語**」
2. 等待 2-3 秒，AI 正在施展魔法...
3. 評語自動出現在「評語」欄位！

> 🎯 **進階技巧**：可以在「手動補充」欄位加入特殊事蹟，AI 會一併納入！

---

### 步驟 5️⃣ 編輯與微調

AI 寫的評語不一定完美，老師可以：

1. 直接點擊「評語」欄位進行編輯
2. 修改不順的句子
3. 加入班級特有的用語
4. 調整語氣和長度

> 💡 **小提示**：點擊「重新生成」可以獲得不同版本的評語！

---

### 步驟 6️⃣ 匯出評語

寫完所有學生後：

1. 點擊右上角「**📤 匯出**」按鈕
2. 選擇格式：
   - **TXT**：純文字，適合複製貼上
   - **Excel**：表格格式，適合存檔
   - **PDF**：美觀排版，適合列印
3. 下載檔案，貼到成績系統！

### 情況 B：需要自行設定

如果沒有得到授權，您需要自行申請：

1. 點擊右上角「**⚙️ 設定**」→「**API Key 設定**」
2. 前往 [Google AI Studio](https://aistudio.google.com/apikey)
3. 使用**個人 Gmail 帳號**登入（⚠️ 不要用教育雲帳號）
4. 點擊「Create API Key」
5. 複製 API Key，貼回我們的系統
6. 點擊「儲存」，完成！

> ⚠️ **重要**：教育雲帳號申請的 API Key 可能無法使用！

## ❓ 常見問題 FAQ

### Q1：生成評語時顯示「配額已用完」？
**A**：免費 API Key 每分鐘有次數限制。請等 1 分鐘後再試，或請管理員授權共享 API Key。

### Q2：為什麼評語生成失敗？
**A**：可能原因：
1. 網路不穩定 → 重新整理頁面
2. API Key 失效 → 重新設定
3. 伺服器忙碌 → 稍後再試

### Q3：可以同時多人使用嗎？
**A**：可以！每位老師的資料都是獨立的，不會互相影響。

### Q4：資料會不見嗎？
**A**：不會！所有資料都存在 Google Firebase 雲端，只要登入同一帳號就能看到。

### Q5：手機可以用嗎？
**A**：可以！完整支援手機和平板，但用電腦會更順手喔。
